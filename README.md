# Аргументы
Переменные и их значения вводятся в формате "x=2 y=3". 

Можно вводить несколько наборов значений переменных. Каждый набор в отдельной строке.

"x=2 y=3

x=3 y=5"

# Операции
Сложение a+b

Вычитание a-b

Умножение a*b

Деление a/b

Натуральный логарифм lna

Степень a^b

Корень n-ой степени a^(1/n)

Отрицание -a
